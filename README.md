# Emissary-Ingress (emissary-ingress)
Emissary-Ingress is a CNCF incubating Kubernetes-native API gateway built on Envoy proxy. It provides ingress control, load balancing, authentication, rate limiting, and traffic management through Kubernetes custom resources.
